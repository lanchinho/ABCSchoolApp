# ABCSchoolApp

## Sobre o Projeto

Este projeto foi desenvolvido para fins de aprendizado, seguindo as aulas do curso **Advanced .NET FullStack: Multi-Tenant Applications** na Udemy, ministrado pelo instrutor Junior Matlou.

## Tecnologias Utilizadas

- **.NET 8**
- **Blazor WebAssembly**
- **C#**
- **MudBlazor** (componentes de UI)
- **Blazored.LocalStorage** (armazenamento local)
- **Toolbelt.Blazor.HttpClientInterceptor** (interceptação de requisições HTTP)
- **Microsoft.AspNetCore.Components.Authorization** (autenticação/autorização)
- **Microsoft.Extensions.Http** (suporte a HTTP)
- **ABCShared.Library** (biblioteca compartilhada interna)

## Componentes e Estrutura

- Componentes customizados em `ABCSchoolAppUI/Components`
- Páginas de autenticação, identidade e multi-tenancy
- Serviços de autenticação, identidade e tenancy em `App.Infrastructure`
- Utilização de dialogs, snackbars e formulários do MudBlazor

## Observações

Este repositório é voltado exclusivamente para estudos e experimentação de conceitos avançados de aplicações multi-tenant com .NET e Blazor.
